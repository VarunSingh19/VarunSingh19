**Radiology & DICOM Management Platform** · Live

Orthanc-backed PACS storage with Redis caching, a real-time Socket.IO chat layer between radiologists and technicians, and a Python image-quality pipeline detecting motion artifacts and exposure defects at **100% classification accuracy**. ### 🛡️ BlackWall
**Enterprise Security Orchestration & Tracking Correlation**

A unified security gateway aggregating six scanning engines (MobSF, ClamAV, TruffleHog, and more) into a real-time React/Next.js dashboard. Includes a **tracker behavioral correlation engine** mapping static SDK signatures against live HTTP traffic to flag trackers as Dormant, Active, or Exfiltrating — plus a dynamic YARA rule-staging pipeline and automated MASVS/DPDP/GDPR compliance mapping.

`React` `Next.js` `Mitmproxy` `YARA` `ClamAV` `Python`
